¿Que es lo que puedo hacer con javascript?







hay que recorda que JS vendria siendo la parte interactiva de la pagina web que estamos creando. Siguiendo la misma analogia podriamos decir que HTML vendria siendo el esqueleto, y CSS seria la ropa y el maquillaje

Es importante recordar que javascript puede usarse inline (que implica colocar el codigo dentro de la misma linea o etiqueta) O con una etiqueta script dentro del archivo html (se coloca una etiqueta script al final del body con todo el codigo que se usará) O linqueado desde un archivo aparte (el cual es el mas usado)
